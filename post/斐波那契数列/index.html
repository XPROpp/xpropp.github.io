斐波那契数列

Jun 18, 2020
1 min read

大家都知道斐波那契数列，现在要求输入一个整数n，请你输出斐波那契数列的第n项（从0开始，第0项为0）。

# -*- coding:utf-8 -*-
class Solution:
    def Fibonacci(self, n):
        # write code here
        if n == 0:
            return 0
        if n ==1:
            return 1
        a =1
        b=0
        ret = 0
        for i in range(0,n-1):
            ret = a + b
            b = a 
            a = ret
        return ret

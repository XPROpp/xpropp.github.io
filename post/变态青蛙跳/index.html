变态青蛙跳

Jun 16, 2020
1 min read

一只青蛙一次可以跳上1级台阶，也可以跳上2级……它也可以跳上n级。求该青蛙跳上一个n级的台阶总共有多少种跳法。

# -*- coding:utf-8 -*-
class Solution:
    def jumpFloorII(self, number):
        # write code here
        # f(n) = f(n-1)+f(n-2)+...+f(1)
        # f(n-1) = f(n-2)+...+f(1)
        # f(n) = 2f(n-1)
        # f(1) = 1 n=1
        if number == 1:
            return 1
        ret = 1
        a = 1
        for i in range(2,number+1):
            ret = 2 * a
            a=ret
        return ret
